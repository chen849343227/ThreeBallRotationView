# ThreeBallRotationView

仿照QQ邮箱的下拉刷新动画

将下面这行代码添加到你项目的的build.gradle里面:

	allprojects {
		repositories {
			...
			maven { url 'https://www.jitpack.io' }
		}
	}Copy
	
然后添加依赖：

	dependencies {
	        compile 'com.github.chen849343227:ThreeBallRotationView:1.0'
	}


参照这篇博客写的，做了一些修改。
